## Getting Started

To get a local copy up and running follow these simple example steps.



### Installation

1. Clone the repo
```sh
git clone git@github.com:sanadwj/Complaint-Management-Portal-backend.git
```

2. Checkout development branch
```sh
git checkout development
```

3. Bundle gems

```sh
bundle install
```

4. Run:

```
rails s -p 5000
```


5. Admin User:

email: `pwc@admin.com`
password: `pwc.admin`
